Current Version: 0.1.0-beta

A lightweight video player that continuously plays random videos or snippets from your video library with optional random start points.

Download Latest Release
Windows 64-bit | No installation required

Quick Start

  1. Download the ZIP file from the releases page
  2. Extract the ZIP file to your desired location
  3. Run GoonPlayer.exe
  4. Add Folders containing your videos in the Settings dialog
  5. Start watching! Your video shuffle begins
No Python installation required - it's a standalone application!

Features

Random Video Selection

Continuously plays videos from your configured folders in random order

Random Clip Mode

Play random-length segments (e.g., 25-45 seconds) from different videos

Random Start Points

Begin each video at a random position for discovery sessions

Persistent Settings

Volume, mute state, and preferences saved across sessions

Fullscreen Mode

Immersive viewing experience with auto-hiding cursor

Keyboard Shortcuts

Quick controls for playback and navigation

Keyboard Shortcuts

Shortcut Action
Spacebar Skip to Next Video
Alt + Enter Toggle Fullscreen
Double-Click Toggle Fullscreen
Esc Exit Fullscreen

Supported Video Formats

MP4, MKV, AVI, MOV, MPEG, WebM, FLV, OGV, VOB, WMV, 3GP, TS, M2TS, MTS, DivX, and more

Format support depends on VLC libraries
