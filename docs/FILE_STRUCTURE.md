# GoonPlayer - Project Structure

## Directory Tree

```
GoonPlayer/
│
├── CORE APPLICATION
│   ├── GoonPlayer.py              ← Main application
│   ├── version.py                 ← Version management
│   ├── GoonPlayer.config.json     ← User config (auto-generated)
│
├── BUILD & DEPLOYMENT
│   ├── scripts/                   ← Build and utility scripts
│   │   ├── build.ps1              ← Build script
│   │   └── make-icon.ps1          ← Icon generation utility
│   ├── requirements.txt           ← Python dependencies
│   ├── .gitignore                 ← Git config
│
├── DOCUMENTATION
│   ├── README.md                  ← Start here (User & Developer Guide)
│   └── docs/
│       ├── INDEX.md               ← Documentation index
│       ├── DEVELOPMENT.md         ← Developer guide
│       └── FILE_STRUCTURE.md      ← This file
│
├── ASSETS
│   ├── app.ico                    ← Application icon
│   ├── icons/                     ← SVG icons (bundled in EXE)
│   │   ├── play.svg
│   │   ├── pause.svg
│   │   ├── volume.svg
│   │   └── ...
│   │
│   └── vlc/plugins/               ← VLC plugins (bundled in EXE)
│       ├── access/
│       ├── demux/
│       ├── video_output/
│       └── ...
│
├── .venv/                         ← Virtual environment
│
└── BUILD OUTPUT (generated by scripts/build.ps1)
    ├── build/                     ← Temp files (.gitignored)
    ├── dist/                      ← Output (.gitignored)
    │   └── GoonPlayer.dist/       ← Standalone build: directory with EXE + deps
    └── release/                   ← Release archives (.gitignored)
        └── GoonPlayer-*.zip       ← Distributable ZIP
```

## Building the Application

### Build the Executable (Standalone)
```powershell
.\scripts\build.ps1
```

Output: `dist\GoonPlayer.dist\GoonPlayer.exe`

### Test the Built Application
```powershell
.\dist\GoonPlayer.dist\GoonPlayer.exe
```

### Create Release Archive
```powershell
.\scripts\build.ps1 -release
```

Output: `release\GoonPlayer-{version}-win64.zip`
